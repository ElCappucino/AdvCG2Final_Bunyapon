This is a project dedicated to DT385 Advanced Computer Graphics 2 subject

The code mainly inspired by

10-Minutes Physics : (https://github.com/matthias-research/pages/blob/master/tenMinutePhysics/)

This project is mainly about implementing everything that i learned in the class including
- PBR image based lighting
- shadow map
- HDR environment map
- Depth of field
- Skeleton Animation
- Softbody Simulation
- 3D smoke simulation
- 2D fire simulation render as billboards

WASD - Move Character
Left Click - Melee Attack
R - Smoke Attack
